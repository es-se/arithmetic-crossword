# Math Crossword 数字ゲーム

スマートフォン向けの **数式クロスワード** です。  
提供されたサンプル画像の **マス配置・空白配置・演算子配置・= の配置を固定** し、**数字だけを差し替えて出題**する構成です。

## 特徴

- サンプル画像と同じ形状を採用
- 数字だけをランダム生成
- スマホ向けのレスポンシブ表示
- タップで数字入力モーダルを表示
- 0〜9 のテンキー入力に対応（2桁まで入力可）
- 難易度 5 段階
- チェック / 入力クリア / 答え表示 あり

## この版の仕様

### 形は固定です

以下は固定です。

- マス配置
- 空白配置
- 演算子配置
- = 配置

数字のみ変化します。

### 難易度

難易度は、**非表示にする数字マス数** で調整しています。

- 簡単: 8 マス非表示
- 普通: 12 マス非表示
- 上級: 16 マス非表示
- 難問: 20 マス非表示
- 超難問: 21 マス非表示

### 計算ルール切替

UI に以下を用意しています。

- 四則演算を優先する（×÷先）
- 左から順に計算する

現在の固定形状では、同一式内で優先順位差が出にくいため、切替の影響は限定的です。

## 使い方

1. `index.html` を開きます
2. 「新しく作る」で新規問題を作成します
3. 空欄マスをタップします
4. モーダルの数字ボタンで値を入力します
5. 「チェック」で確認します

## GitHub Pages で公開する手順

1. GitHub に新しいリポジトリを作成します
2. この ZIP を展開してファイルをアップロードします
3. リポジトリの **Settings → Pages** を開きます
4. Branch を選択して公開します

## 補足

- 盤面生成は `app.js` 内で行っています
- 数字生成は固定形状の方程式群を満たすように構成しています
- 生成された盤面は `チェック` ボタンで入力一致数と成立式数を確認できます
