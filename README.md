# Math Crossword 数字ゲーム

スマートフォン向けの **数式クロスワード** です。  
提供されたサンプル画像の **マス配置・空白配置・演算子配置・= の配置を固定** し、数字だけを差し替えて出題する構成です。

## 特徴

- サンプル画像と同じ形状を採用
- 数字だけをランダム生成
- スマホ向けのレスポンシブ表示
- タップで数字入力モーダルを表示
- 0〜9 のテンキー入力に対応（2桁まで入力可）
- 難易度 5 段階
- チェック / 入力クリア / 答え表示 あり

## この版の仕様

- **形は固定**です
  - マス配置
  - 空白配置
  - 演算子配置
  - `=` 配置
- **数字のみ変化**します
- 難易度は、非表示にする数字マス数で調整しています

## 使い方

1. `index.html` を開きます
2. 「新しく作る」で新規問題を作成します
3. 空欄マスをタップします
4. モーダルの数字ボタンで値を入力します
5. 「チェック」で確認します

## GitHub Pages で公開する手順

1. GitHub に新しいリポジトリを作成します
2. この ZIP を展開してファイルをアップロードします
3. リポジトリの **Settings → Pages** を開きます
4. Branch を選択して公開します

## 補足

- 計算ルール切替は UI に残しています
- 現在の固定形状では、演算子配置の都合上、切替の影響は限定的です
- 盤面生成は `app.js` 内で行っています
